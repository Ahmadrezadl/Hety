# GitHub Repository Description

All-in-one desktop developer cockpit for SSH terminals, Git workflows, and PostgreSQL databases.

## Longer Description

Hety brings project-based SSH sessions, Git repository controls, and PostgreSQL query tools into one local desktop workspace built with Electron, React, TypeScript, Vite, and Tailwind CSS.

electron, react, typescript, vite, tailwindcss, ssh, git, postgresql, developer-tools, desktop-app
